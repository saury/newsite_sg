诚招供应商

一、手机供应商：
  1. 产品符合国外市场需求：三码机、五码机、品牌机均可 
  2. 有一定实力，能提供有竞争力的价格 
  3. 紧跟市场最新产品
三、家居产品供应商（卫浴，家装，油画，床上用品，宠物用品，厨房用品，户外园林用品，小家电等）：
  1. 有产品外销经验,有意开拓国际市场的 
  2. 对市场有良好的把握,有较强的产品开发和采购能力 
  3. 能提供备货
五、时装供应商： 
  1. 产品主要出口欧美市场，或符合欧美审美风格 
  2. 款式多，能理解及接受多品种小批量的电子商务营销模式 
  3. 交期及时，能按需生产。最好有适量的库存现货
七、 球服，家居服类供应商：
  1. 产品齐全，货源稳定 
  2. 市场反映灵敏，具备一定的新品开拓能力 
  3. 有外销经验更佳
九、车载供应商：
  1. 资金实力雄厚，能提供账期，在汽车市场的贸易商或工贸一体供应商 
  2. 资质齐全 
  3. 本身具有快速，稳定的供货链和快速的搜货能力，能供应多品种，小批量产品，并有拓展供应链的需求和意向 
  4. 注重品质，能提供优质的售后服务 
  5. 本身有从事外贸业务的优先
十一、新奇特等器具供应商：
  1. 对电视购物及创意概念的产品有相当的了解及搜寻能力 
  2. 贸易商最好有备货和仓库 
  3. 产品种类丰富，跨品类
十三、玩具、节日用品类供应商：
  1. 有产品外销经验 
  2. 有节日类产品开拓能力 
  3. 能提供现货
二、COSPLAY或COSTUME供应商：
  1. 必须能够根据客户的要求（尺码、三围、体重等）定做服装 
  2. 提供服装的同时能够提供动漫人物的配件周边 
  3. 依据我们调研后的市场新品图片做出相似度90%以上的产品 
  4. 产品种类丰富多样，保证质量的同时价格合理
四、婚纱礼服供应商：
  1. 我公司每月采购，要求供应商具有独立工厂 
  2. 生产外销婚纱、礼服2年以上经验 
  3. 有自己的管理梯队，专业的质检团队，稳定的面料辅料供应商 
  4. 具备新品开拓能力，对小批量定制有一定了解
六、童装供应商：
  1. 产品出口欧美市场 
  2. 款式新颖，可爱，适合国外父母需求 
  3. 必须了解欧美市场童装的要求，对出口童装原材料要求严格 
  4. 能提供有竞争力的价格
八、电脑，安防，手机配件等供应商：
  1. 资金实力雄厚，贸易商或工贸一体供应商 
  2. 本身具有快速，稳定的供货链和快速的搜货能力，能供应多品种，小批量产品，并有拓展供应链的需求和意向 
  3. 注重品质，能提供优质的售后服务 
  4. 本身有从事外贸业务的优先
十、MP3, 游戏机等 供应商：
  1. 产品齐全，尽量提供多种类产品和配件，货源稳定 
  2. 主机和配件可以一起供货，此项为必要条件 
  3. 服务好，配合及时到位 
  4. 具有开拓精神，市场反应敏锐，新品供货能力强
十二、首饰，配饰供应商：
  1. 产品符合国外客户审美要求 
  2. 款式时尚,设计精美，紧跟国际时尚品味 
  3. 能提供有竞争力的价格 
  4. 能在规定时间内准时交货， 有库存
